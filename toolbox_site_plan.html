Key Features

What makes this site stand out

🔒 Privacy First

Processing happens in your browser using WebAssembly & Canvas API. Files never leave your device for most tools. Those that need server-side processing auto-delete within 1 hour.

⚡ Batch Processing

Upload 50+ files at once. Apply the same settings to all. Download as a single ZIP. Progress bar per file and overall. Works even on slow connections.

👁️ Live Preview

Every tool shows a before/after split view as you adjust settings. No surprises on download. Compare original vs output size in real-time.

📱 Works Everywhere

Fully responsive — desktop, tablet, mobile. PWA installable. Offline capable for browser-based tools. Share tool URLs directly for quick access.

🤖 AI-Powered Tools

Background remover, image upscaler, PDF OCR, and auto-tag generator are powered by on-device ML models (ONNX Runtime) — no cloud API costs for basic tiers.

📋 Tool History

Browser-local history of recent conversions. One-click re-process with same settings. Export your workflow as a recipe to share with teammates.
